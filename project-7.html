App Overview

The Movie Search App is a web application developed to provide users with a convenient way to search for movies, view details, and manage their watchlist. With a user-friendly interface, it allows users to enter the name of a movie, fetches information from an API, and displays key details such as title, photo, release date, and a brief description of the movie. Additionally, users can rate movies with star ratings and save their favorite movies to a watchlist stored in local storage. The app also provides a watchlist page where users can access their saved movies and remove them from the list if desired.

Users can add movies to their watchlist with a simple click, and the selected movies are saved in local storage for future reference. The watchlist page allows users to view their saved movies and provides a remove button next to each movie, enabling easy removal from the watchlist.
